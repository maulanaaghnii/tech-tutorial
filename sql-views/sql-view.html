SQL View

Introduction

View dalam SQL adalah tabel virtual yang tersimpan dalam database dengan nama tertentu. Ini sebenarnya adalah komposisi dari sebuah tabel dalam bentuk query SQL yang telah ditentukan sebelumnya. Sebuah view dapat berisi baris dari tabel yang ada (semua atau yang dipilih). View dapat dibuat dari satu atau banyak tabel. Kecuali diindeks, view tidak ada secara fisik dalam database.


Untuk membuat view dalam database, Anda dapat menggunakan pernyataan SQL CREATE VIEW.


CREATE VIEW view_name AS
SELECT column1, column2....
FROM table_name
WHERE [condition];

Untuk memperbarui view dalam database, Anda dapat menggunakan pernyataan SQL UPDATE.


UPDATE view_name
SET column1 = value1, column2 = value2...., columnN = valueN
WHERE [condition];

Untuk menghapus view dalam database, Anda dapat menggunakan pernyataan SQL DROP VIEW.


DROP VIEW view_name;

Pernyataan RENAME TABLE dalam database MySQL digunakan untuk mengubah nama view. Anda hanya perlu memastikan bahwa nama baru view tidak bertabrakan dengan nama view yang sudah ada.


RENAME VIEW old_view_name TO new_view_name;
